Bootcamp 2020: Pendulum Array

My first programming project, concluding the 2 week Programming Bootcamp at City (2020). This Processing project demonstrates a comprehensive understanding of the use of arrays, loops, randomness and classes and objects. I created a simple graphic using arrays of balls. Both arrays travel against each other until they hit the wall, at which point they 'bounce' and travel in the opposite direction. The coding includes the feature of randomization. Therefore, each time the program is run, the balls will be of different colours.
